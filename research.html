Moving Path Following for a single robotic vehicle

A part of my doctoral studies involved investigation of the Moving Path Following (MPF) motion control problem for underactuated vehicles such as unicycle, AUVs, ASVs and fixed wing UAVs. In the MPF problem, the robotic vehicle is tasked to following a geometric path that is specified with respect to a moving frame of reference. Concepts of geometric control, nonlinear model predictive control, and robust control have been used to develop MPF control laws that guarantee stability in presence of uncertainties in the inner-loop autopilot controllers. Experiments were performed to validated the proposed controllers. This effort led to development of cooperative moving path following method that is described below.

Cooperative Moving Path Following using Event based Communication

This research effort focussed on development of a novel cooperative control framework termed the "Cooperative Moving Path Following", wherein robotic vehicles are tasked to follow a specified geometric path that is moving with respect to a target, while achieving a desired cooperation objective. This is in contrast to the existing Cooperative Path Following framework, where the robotic vehicles perform a coordinated maneuver to follow a fixed reference path. The results on the path following controllers are extended to incorporate the effects of moving frame of reference of the path, leading to a Lyapunov-based nonlinear control law for a moving path following motion control problem of an individual robotic vehicle. In order to achieve cooperation, the moving path following controllers are augmented with decentralized, first order consensus approach, resulting in a Cooperative Moving Path Following architecture. Furthermore, static and dynamic event-based control and communication methods are applied at the cooperation level to reduce the frequency of information exchange between the robotic vehicles. The advantage of the CMPF method proposed lies in its flexibility to choose different parameterizable paths and is illustrated in simulations through coordinated source seeking and convoy protection scenarios using under-actuated robotic vehicles.

Cooperative Path Following using Event based Communication

This research effort aimed to propose a novel approach to the cooperative path following (CPF) control problem with an objective to reduce the frequency of communication between the robotic vehicles while achieving cooperation. Specifically, a decentralized, event-based cooperative controller is designed to achieve coordination between the robotic vehicles, tasked to follow a priori specified reference geometric path while maintaining a desired formation. The event-based cooperative controller transmits the necessary information at discrete event times, that are computed based on an event triggering condition designed such that the convergence and stability properties of the consensus controller is preserved. The Input-to-State Stability framework is utilized to prove the stability and convergence of the CPF control with event-based control and communication scheme. The proposed method is experimentally validated using Autonomous Surface Vehicles communicating over Wi-Fi. It is shown that the event-based approach results in significant reduction of information exchange between the vehicles when compared to the conventional periodic transmission.

Localization of an acoustic source using TDOA measurements

This project was in collaboration between University of Porto, NTNU Norway and KTH Sweden. The idea was to estimate the position of an acoustic fish-tag using only the Time-of-Arrival measurements of the acoustic signal transmitted by the fish-tag. To this end, three ASVs mounted with acoustic receivers were used and the objective was to estimate the position of the fish-tag and track it using ASVs. I was responsible to develop estimators to localize the acoustic source (fish-tag). Two types of estimators were developed namely, Extended Kalman Filter and eXogenous Kalman Filter. Both the estimators were implemented in C++ and experimentally validated. Another parallel work included use of NMPC in closed loop with Extended Kalman Filter to estimate and track an unknown source using single robotic vehicle. The key idea was to use economic cost within the NMPC formulation to generate trajectories for the robotic vehicle such that the source position is observable. This was a simulation study.

Transportation of cable suspended load using quadrotor

In my masters thesis, I designed and implemented a Linear MPC for transportation of cable suspended load using quadrotor. This included simulation study and subsequent implementation of the MPC for quadrotor using the Paparazzi control software. The key component of my thesis was to develop a software framework in C that executes the MPC controller on an off-board computer and communicates the control commands to the quadrotor. Additionally it acquires the quadrotor telemetry data thereby forming a networked feedback control system.
